# Geometric Sequences

## Definition

A geometric sequence has a common ratio r between successive terms and:
- Recursive definition: a<sub>n</sub> = r∙a<sub>n-1</sub> with a<sub>0</sub>
- Closed formula: a<sub>n</sub> = a<sub>0</sub>∙r<sup>n</sup>

### Examples

- 3, 6, 12, 24, 48, ...
- 27, 9, 3, 1, 1/3, ...
