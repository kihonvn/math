# The Power Set of A Set

## Definition

If A is any set, the power set of A is the set containing all subsets of A, denoted P(A).

### Examples

- A = {1, 2} ⇒ P(A) = {∅, {1}, {2}, {1, 2}}

## Theorems

### Cardinality of the power set of a set

|P(A)| = 2<sup>|A|</sup>

## Calculator syntax

`Subsets[{...}]`

### Examples

- `Subsets[{1,2,3}] = {{}, {1}, {2}, {1, 2}}`
