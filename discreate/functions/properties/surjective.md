# Surjective Functions - Surjections - Onto Functions

## Definition

A function &#x1D453; : &#x1D44B; &#x2192; &#x1D44C; is surjective if its range, &#x1D453;(&#x1D44B;), is equal to its codomain, &#x1D44C;.
