# The Universal Quantifier

## Definition

If p is a proprosition over U with T<sub>p</sub> = U, said "For all n in U, p(n).", denoted (∀n)<sub>U</sub>(p(n)) or, in short, (∀n)(p(n)). The symbol ∀ is the universal quantifier.
